Used to convert values to numeric first replace ',' , '%' so that they can be converted to number then values will be converted to numeric and invalid values become NaN. A value can't be converted to numeric if it contains non number such as text, symbols, mix of text and number(12k) These are not pure numbers, so pd.to_numeric() cannot convert them unless cleaned. so errors='coerce' is used to handle invalid values while converting to numeric.

In[8]:
times['num_students'] = times['num_students'].str.replace(',', '')
times['num_students'] = pd.to_numeric(times['num_students'], errors='coerce')

times['international_students'] = times['international_students'].str.replace('%','')

Q3 = latest['total_score'].quantile(0.75)

lower = Q1 - 1.5 * IQR
upper = Q3 + 1.5 * IQR

outliers = latest[(latest['total_score'] < lower) | (latest['total_score'] > upper)]
